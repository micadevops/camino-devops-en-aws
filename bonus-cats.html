🐱 Bonus · Cat API

Llamando a una API externa

El frontend hace un fetch a The Cat API y muestra la respuesta JSON en pantalla — sin backend propio.

Request que hace el browser
GEThttps://api.thecatapi.com/v1/images/search
  ?has_breeds=true&limit=1
  &x-api-key: DEMO-API-KEY
🐱
Presioná el botón para cargar un gatito

  


  


  

  
  

    
Lo que está pasando

    
    
      
  • El browser hace un GET request a The Cat API
    • 
      
  • La API devuelve un JSON con la URL de la imagen y datos de la raza
    • 
      
  • El frontend parsea el JSON y renderiza la imagen y los datos
    • 
      
  • No hay backend propio — el frontend habla directo con la API externa
    • 
      
  • El API key va en el header — así la API sabe quién está pidiendo
    •
